# Заметки (Reference)

Справочная информация, которая не требует никаких действий, но которую важно сохранить.

Примеры: документы, ссылки, расписания, инструкции.

## Инструкция по работе (GTD)
**Цель:** Хранение справочной информации.
**Инструкция:** Складывайте сюда то, что не требует действий, но может пригодиться (документы, ссылки, инструкции). Организуйте их по категориям (личное, работа и т.д.) для удобного поиска.
