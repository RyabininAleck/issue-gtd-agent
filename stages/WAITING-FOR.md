# Список ожидания (Waiting For)

Предназначен для задач, которые делегированы другим людям.

Для каждой задачи фиксируйте:
- кому поручено;
- дату поручения;
- ожидаемый результат.

Это нужно, чтобы регулярно напоминать исполнителям и отслеживать статус.

## Инструкция по работе (GTD)
**Цель:** Контроль делегированных задач.
**Инструкция:** Если задачу делает другой человек, запишите: кто должен сделать, что именно и дату поручения. Раз в неделю просматривайте список и «пинайте» (напоминайте) ответственных.
