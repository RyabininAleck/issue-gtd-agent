# Проекты (Projects)

Список всех многошаговых задач (любое дело, требующее более одного шага).

Для каждого проекта обязательно определите:
- критерий завершения;
- примерный план;
- первый шаг, который отправляется в работу.

## Инструкция по работе (GTD)
**Цель:** Управление многошаговыми задачами.
**Инструкция:** Любое дело, требующее более одного шага — это проект. Для каждой карточки здесь обязательно пропишите:
- Критерий завершения (как выглядит готовый результат).
- Примерный план.
- Первый конкретный шаг (который должен уйти в «Текущие действия»).
